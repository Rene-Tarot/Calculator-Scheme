# Calculator-Scheme

This is a simple calculator made in Scheme.

## How to run

1. Download [Racket](https://racket-lang.org/)
2. Open the file in the IDE.
3. Select the language as Pretty Big.

## Operation Format

*operation* number number

- **+**: Addition.
- **-**: Substraction.
- **\***: Multiplication.
- **/**: Divition.
- **sqr**: Exponential
- **sqrt**: Square Root.
- **%**: Remainder.
- **div**: Residue
- **fact!**: Factorial.
- **fibonacci**: Fibonacci
